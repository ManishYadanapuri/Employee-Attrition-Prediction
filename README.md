# 👔 Employee Attrition Prediction

An end-to-end Machine Learning project to predict employee attrition.

## 📊 Project Overview
- **Problem:** Predict which employees are likely to leave the company
- **Dataset:** IBM HR Analytics (1,470 employees, 31 features)
- **Best Model:** XGBoost (ROC-AUC: ~87%)

## 🛠️ Tech Stack
Python | Pandas | Scikit-learn | XGBoost | SMOTE | Streamlit | Joblib

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py

## 📈 Top Attrition Drivers Found
- OverTime
- Monthly Income
- Job Satisfaction
- Distance from Home
- Work Life Balance
